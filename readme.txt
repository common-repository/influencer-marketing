Influencer Marketing & Press Release Engagement Software System (IMPRESS) for companies & brands or authority marketing.

== Description ==
*Heads up! Portions of this plugin require a paid http://SwiftCRM.Com account to be of use.*

Some of the basics are free, and this plugin is useful without any SwiftCloud account at all to generate and format press releases and add a press section to your website. You can also do some basic influencer marketing with it, and of course all of this is perfect for either authority marketing (authors, thought leaders, etc.), brands and companies, and entertainers (musicians, actors etc.) who work with conventional media outlets.

**This system will... (free features)**

1. Auto-generate 3x pages: Press Releases, Press Clippings, and Press Kit
2. Create a custom post type for Press Releases, formatted per industry standards
3. On the backend, it adds a dashboard to monitor the media of journalists and influencers you care about (RSS, Twitter, etc.)
4. There's a basic Press Release Wizard for people who have never written a release (note: run it by an editor before sending!)
5. Guest Blogging tools will help focus on most important most relevant blogs to partner with

**Paid Features...**

1. via SwiftCRM.Com, optionally get automated press clippings (in private beta; paid feature)
2. via SwiftCRM.Com syndicate your releases to media outlets and influencers
3. Need something else? Just ask. We listen.

For the moment we are most concerned with real-world-value more than money - so if you need something, drop us a line at https://SwiftCRM.Com/support - we want to help make this useful.

More details about the entire system are available at http://SwiftMarketing.com/IMPRESS

== Installation ==
You probably know the drill by now, but if not, here's a step by step suggestion.

1. Upload the `influencer-marketing` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. It will ask to auto create pages for press, media kit and clippings.
